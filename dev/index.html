BaseModelica.jl: BaseModelica Importers for ModelingToolkit.jl

BaseModelica.jl is an interface for the SciML open source software organization for importing BaseModelica, a new standard from the Modelica association for lowered Modelica code. As such, it makes it easy to import Modelica models into ModelingToolkit.

Installation

To install BaseModelica.jl, use the Julia package manager:

using Pkg
Pkg.add("BaseModelica")

API

BaseModelica.parse_basemodelica — Function
parse_basemodelica(filename::String)::ODESystem

Parses a BaseModelica .mo file into a ModelingToolkit ODESystem.

Example

parse_basemodelica("testfiles/NewtonCoolingBase.mo")
